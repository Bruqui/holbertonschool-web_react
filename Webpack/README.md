# Webpack

Learning how to bundle a front-end application with Webpack.

## Tasks

- **task_0** — Basic setup: Webpack via the CLI, no config file.
- **task_1** — Using a `webpack.config.js` file with jQuery and Lodash.
- **task_2** — Adding CSS and images to the bundle (loaders).
- **task_3** — Dev server, code splitting into modules, and tree shaking.

## Usage

Each task is a standalone folder. From inside a task:

```bash
npm install
npm run build      # build into public/ (or dist/ for task_0)
npm run start-dev  # task_3 only: dev server on http://localhost:8564/
```
